# Exercise in construcing integer codes using primitive recursive functions

What it says on the tin, this is WIP, currently one of the tests is failing, so use at own risk.

I can't say this is the cleanest Haskell code, or performant for anything, it's a sanity check on a pen-and-paper construction in a cartesian (closed) category with a natural number object, à la <https://ncatlab.org/nlab/show/natural+numbers+object#withparams>.
